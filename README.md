# ArtEvoke - GitHub Pages

This repository hosts the public-facing website for the ArtEvoke project at [augustovillar.github.io](https://augustovillar.github.io).

## About

ArtEvoke is an innovative platform that connects art and memory through AI, designed to support non-drug interventions for people living with Alzheimer's disease.

The website features:
- **Home Page**: Introduction to the platform and its two main modes (Memory Reconstruction and Art Exploration)
- **About Page**: Detailed information about the research, development, and team behind ArtEvoke

## Technology

This is a static website built with pure HTML, CSS, and JavaScript, hosted on GitHub Pages.

## Local Development

To view the site locally, simply open `index.html` in your web browser.

## Deployment

The site is automatically deployed via GitHub Pages from the main branch.
